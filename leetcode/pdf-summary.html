多种概率分布总结

一. 离散概率分布

1. 两点分布（伯努利分布、0-1分布）

  • 两点分布也称为0-1分布、伯努利分布，随机变量只取0或1，如一次抛硬币实验的正反面。概率质量函数：
P(X=x)={px(1−p)1−xforx=0or10otherwiseP(X=x)=\begin{cases} { p }^{ x }{ (1-p) }^{ 1-x }\quad for\quad x=0\quad or\quad 1 \\ 0\quad \quad \quad \quad \quad \quad otherwise \end{cases}P(X=x)={px(1−p)1−xforx=0or10otherwise​
  • 期望、方差
E(X)=p,Var(X)=p(1−p)E(X) =p,\quad Var(X) = p(1-p)E(X)=p,Var(X)=p(1−p)
  • 应用：极大似然估计（MLE）

2. 二项分布

  • 二项分布是进行n次独立实验，每次实验都是一次两点分布，随机变量X表示n次实验成功次数。概率质量函数：

  • 期望、方差
E(X)=np,Var(X)=np(1−p)E(X)=np,\quad Var(X)=np(1-p)E(X)=np,Var(X)=np(1−p)

3. 几何分布

  • 几何分布也是进行多次独立两点实验，随机变量X表示第一次成功所进行实验的次数。概率质量函数：

  • 负二项分布是几何分布的一般形式，随机变量X表示直到成功r次所进行实验的次数。概率质量函数：

5. 超几何分布

  • 随机变量X表示(在N个物品中有指定物品M个)不放回抽取n次，抽中指定物品的个数。概率质量函数：

6. 泊松分布

  • 泊松分布适合于描述单位时间内随机事件发生次数的概率分布。概率质量函数：
P(X=k)=e−λλkk!P(X=k)=\frac { { e }^{ -\lambda }{ \lambda }^{ k } }{ k! }P(X=k)=k!e−λλk​

参数λ是单位时间（或单位面积）内随机事件的平均发生率。

  • 期望、方差

  • 在二项分布的伯努利试验中，如果试验次数n很大，二项分布的概率p很小，且乘积λ= np比较适中，则事件出现的次数的概率可以用泊松分布来逼近。https://zh.wikipedia.org/wiki/泊松分佈
  • λ大概等于20时，泊松分布基本可以近似为正态分布进行处理

二. 连续概率分布

1. 均匀分布（矩形分布）

  • 均匀分布是对称概率分布，在相同长度间隔的分布概率是等可能的。概率密度函数：
